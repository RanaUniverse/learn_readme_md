# Module Details

## Command Handlers
- `start_module.py`: Handles the `/start` command.
- `help_module.py`: Handles the `/help` command.

## Conversation Handlers
- `note_making_conversation.py`: Implements the logic for creating notes through a conversation.

## Database Modules
- `database.py`: Contains the database connection and session logic.
- `models.py`: Defines the data models (e.g., `User`, `Note`).

## Message Handlers
- `text_msg_module.py`: Processes text messages sent by users.
